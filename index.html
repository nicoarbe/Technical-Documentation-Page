Visión general

Una visión general de lo que Bulma como marco se trata

Empezar. Sólo necesita 1 archivo CSS para utilizar Bulma.

Hay varias maneras de comenzar con Bulma. Puede:

  1. Utilizar npm para instalar el paquete Bulma
  2. Utilizar la CDN jsDelivr para enlazar a la hoja de estilo de Bulma
  3. Utilizar el repositorio GitHub para obtener la última versión de desarrollo

Requisitos del código. Para que Bulma funcione correctamente, debe hacer que su página web sea responsiva.

  1. Utilice el doctype HTML5
    2. <!DOCTYPE html>
  2. Añadir la metaetiqueta de ventana gráfica responsiva
    3. <meta name="viewport" content="width=device-width, initial-scale=1">

Clases CSS. Bulma es simplemente una colección de clases CSS. Escriba el código HTML que desee.

Bulma es un framework CSS, lo que significa que el resultado final es simplemente un único archivo: https://github.com/jgthms/bulma/blob/master/css/bulma.css .css

Personalizar

Crea tu propio tema con un simple conjunto de variables

Conceptos. Lo que hace que Bulma sea personalizable.

Bulma es altamente personalizable gracias a 419 variables Sass que viven en 28 archivos. Estas variables existen en 4 niveles:

  • variables iniciales: variables globales con valores literales
  • variables derivadas:variables globales con valores que hacen referencia a otras variables, o se calculan
  • variables genéricas:para los elementos HTML que no llevan ninguna clase CSS
  • variables de elemento/componente:variables específicas de un elemento/componente Bulma

Puesto que estas variables llevan el indicador, se les puede asignar un nuevo valor antes o después de haber sido importados. !default

Columnas

El poder de Flexbox en una interfaz sencilla

Básico

Construir un diseño de columnas con Bulma es muy simple:

  1. Añadir un contenedor <columns>
  2. Agregue tantos elementos como desee column

Cada columna tendrá el mismo ancho,independientemente del número de columnas.

Diseño

Diseñe la estructura de su página web con estas clases CSS

Contenedor. Un contenedor simple para centrar el contenido horizontalmente

Es un elemento de utilidad simple que le permite centrar el contenido en ventanas gráficas más grandes. Se puede utilizar en cualquier contexto, pero sobre todo como un hijo directo de uno de los siguientes: container

  • navbar
  • hero
  • section
  • footer

Formulario

Los controles de formulario indispensables, diseñados para la máxima claridad

Bulma admite los siguientes elementos de formulario HTML nativos: y . form button input textarea label

Se admiten las siguientes clases CSS:

  • label
  • input
  • textarea
  • select
  • checkbox
  • radio
  • button
  • help

Para mantener el diseño equilibrado de forma uniforme, Bulma proporciona un contenedor muy útil con el que puede ajustar los controles de formulario. Al combinar varios controles en un formulario,utilice la clase como contenedorpara mantener el espaciado coherente. control field

Componentes

Componentes avanzados de varias piezas con muchas posibilidades

Miga de pan. Un sencillo componente de miga de pan para mejorar tu experiencia de navegación

La ruta de navegación Bulma es un componente de navegación simple que solo requiere un contenedor y una lista. Los divisores se crean automáticamente en el contenido del pseudo-elemento de las etiquetas. breadcrumb ul ::before li

Puede informar a la página actual utilizando el modificador de una etiqueta. Desactivará la navegación de enlaces internos. is-active li
